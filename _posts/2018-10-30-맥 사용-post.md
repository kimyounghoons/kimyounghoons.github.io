## 환경변수 설정

bash_profile 을 열어준다.

{% highlight css %}
nano ~/.bash_profile
{% endhighlight %}
 
 bash_profile 에 alias 설정을 해주면 code로 앱을 실행 시킬수 있다.

{% highlight css %}
alias code="open -a /Applications/Visual\ Studio\ Code.app/"
{% endhighlight %}
